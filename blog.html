Important FAQ

What is the Document Object Model (DOM)?

The DOM is a programming interface that represents an HTML or XML document as a hierarchical tree structure. It allows JavaScript to access, modify, and manipulate elements, attributes, and styles dynamically.

How do you select an element from the DOM?

You can select elements using JavaScript methods:

  • document.getElementById("id") – Selects an element by its ID
  • document.getElementsByClassName("class") – Selects elements by class name.
  • document.getElementsByTagName("tag") – Selects elements by tag name.
  • document.querySelector("selector") – Selects the first matching element.
  • document.querySelectorAll("selector") – Selects all matching elements
What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a parent element handles events for its child elements using event bubbling. Instead of attaching listeners to multiple child elements, a single listener on the parent can manage them all. This improves performance, reduces memory usage, and makes it easier to handle dynamically added elements.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes and styles using JavaScript:


  • Attributes: Use element.setAttribute("attr", "value") to modify, element.getAttribute("attr") to retrieve, and element.removeAttribute("attr") to remove an attribute.

  • Styles: Modify inline styles using element.style.property = "value" – Sets the value of a CSS property. (e.g., element.style.color = "red")

  • Classes: Use classList.add("class"), classList.remove("class"), classList.toggle("class") to modify element classes dynamically.
